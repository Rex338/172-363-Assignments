  2. Course
    Catalogue Description

    Introduction to project management concepts, tools, and techniques: integration management and project planning, scope management, scheduling, budget control, human resource management, communication management, risk analysis and management, project quality management, and procurement management.

    3. Textbook

    “Information Technology Project Management”

    by Kathy Schwalbe, Course Technology, 8th Ed, 2015 (or the 7th edition)

    Course Objectives

    The course objectives are to

    1. To apply all appropriate project management techniques. [PO #1]
    2. To learn how to work in teams. [PO #4]
    3. To enhance communication and writing skills. [PO #4]
    4. To instill life-long learning skills. [PO #5]
    Course Learning Outcomess

    Upon completion of this course, students will have the ability to:

    1. Estimate the cost of the project and perform risk management and configuration management in software projects.
    2. Measure project progress, productivity and other aspects of the software process.
    3. Work in a team to develop project management plan.
    4. Draft contracts and proposals for software projects.
    5. Develop a project plan for a significant development effort.
    6. Self-learn new tools that help in managing a software project.
